# burn-out-rate-predictor
Script developed to analyse and predict employees' burn out rate based on [kaggle dataset](https://www.kaggle.com/datasets/blurredmachine/are-your-employees-burning-out?select=train.csv) for the subject "INE5649 - Técnicas Estatísticas de Predição" at the Federal University of Santa Catarina.

Authors: Amabyle Heck, Gustavo Schmitz Albino, and Jackson Dener Wrublack.
